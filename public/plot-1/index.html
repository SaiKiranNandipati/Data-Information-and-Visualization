Plot 1:Regression Line Visualizer

This interactive tool allows you to explore 2D regression analysis by adding and removing points on a plane.

To add a point, simply click on an empty area of the plane. The regression line will be automatically recalculated and displayed based on the new point.

If you want to remove a point, click on an existing point, and the regression line will update accordingly.

If you want to start fresh, you can click the "Reset" button to clear all points from the plane.
