Plot 3: Collapsible Tree Showing file structure of any Github Repo

This visualization displays a collapsible tree structure representing the folder hierarchy of a GitHub repository.

In this visualization, blue circles represent directories that can be expanded or collapsed, while files are represented by blue circles.

If you wish to visualize a different repository, input the repository owner and name in the fields below and click "Fetch Data".

Please note that fetching data may take some time as it involves multiple requests to the GitHub Contents API.

Once the new visualization is loaded, clicking "Save Data" will replace the default structure being displayed.
